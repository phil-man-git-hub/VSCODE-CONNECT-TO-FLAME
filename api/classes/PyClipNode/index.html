    Class: PyClipNode

    Module: flame

    Inherits from: PyNode, PyFlameObject, instance, object

    Description

    Class derived from PyNode. This class represents a Clip node.

    Properties

    Name Description
    attributes The attributes of a python object.
    clip The associated PyClip or PySequence of the clip node.
    input_sockets Return a list of the node input sockets names.
    output_sockets Return a list of the node output sockets names.
    parent The parent object of this object.
    sockets Return a dictionary of the input/output sockets names and their connections.
    version_uid The clip node's list of available version unique IDs.
    version_uids The clip node's list of available version unique IDs.
