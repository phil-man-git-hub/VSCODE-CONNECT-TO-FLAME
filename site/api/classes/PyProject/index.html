It will use the default colour management shared path if empty. overwrite_existing -- Specifies if the export should overwrite an existing OCIO config with the same name located in the same destination_folder. export_as_locked -- Specifies if the exported OCIO config should be locked (the 'LockedPolicy' parameter inside the settings.cfg sidecar file). generate_ocioz -- Specifies if an OCIOZ archive should be created alongside the exported OCIO config.
