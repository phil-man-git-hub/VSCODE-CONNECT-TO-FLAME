# Function: clear_unreferenced_cache

**Module**: `flame`

**Signature**: `clear_unreferenced_cache(...)`

clear_unreferenced_cache([  (bool)all_projects=False]) -> None :
    Clear cached media that is no longer referenced by any clip in eitherthe current project or any other project.
    
    all_projects -- If True, will also clear unreferenced cached media from all other accessible projects.
                    Default is False, which will only clear unreferenced cached media from the current project.
